#  ISO 13485:2016 – Medical Devices Quality Management System Checklist

**Purpose:**  
ISO 13485 specifies requirements for a QMS applicable to the design, production, installation, and servicing of medical devices, ensuring product safety and regulatory compliance.

---

| Clause | Requirement | Description | Status (✓/✗) | Remarks |
|---------|--------------|-------------|---------------|---------|
| 4.1 | QMS Requirements | Establish and maintain a documented QMS. |  |  |
| 4.2 | Documentation | Maintain quality manual, procedures, and records. |  |  |
| 5.1 | Management Commitment | Ensure leadership commitment to product safety and quality. |  |  |
| 5.3 | Quality Policy | Define and communicate a quality policy. |  |  |
| 6.2 | Human Resources | Ensure competence of personnel performing device-related work. |  |  |
| 7.1 | Planning | Plan device realization processes. |  |  |
| 7.3 | Design & Development | Establish design controls for medical devices. |  |  |
| 7.4 | Purchasing | Evaluate and control suppliers. |  |  |
| 7.5 | Production & Servicing | Control production and installation processes. |  |  |
| 7.6 | Calibration | Maintain control over monitoring and measuring equipment. |  |  |
| 8.2 | Feedback & Complaints | Collect and analyze post-market feedback. |  |  |
| 8.3 | Nonconforming Product | Identify and control defective devices. |  |  |
| 8.4 | Analysis of Data | Evaluate data from production and complaints. |  |  |
| 8.5 | Improvement | Implement corrective and preventive actions (CAPA). |  |  |
