# ISO 45001:2018 – Occupational Health & Safety Management System (OHSMS) Checklist

**Purpose:**  
ISO 45001 outlines requirements for an OHS management system to improve employee safety, reduce workplace risks, and create healthier work environments.

---

| Clause | Requirement | Description | Status (✓/✗) | Remarks |
|---------|--------------|-------------|---------------|---------|
| 4.1 | Context | Identify internal/external OHS issues. |  |  |
| 4.2 | Workers & Interested Parties | Identify workers’ needs and expectations. |  |  |
| 5.1 | Leadership | Demonstrate leadership commitment to safety culture. |  |  |
| 5.2 | OH&S Policy | Establish and communicate OHS policy. |  |  |
| 6.1 | Hazards & Risks | Identify hazards and assess OHS risks/opportunities. |  |  |
| 6.2 | Objectives | Define OHS objectives and plans to achieve them. |  |  |
| 7.2 | Competence | Ensure worker competence in OHS procedures. |  |  |
| 7.3 | Awareness | Promote awareness of OHS responsibilities. |  |  |
| 7.4 | Communication | Ensure OHS information is effectively communicated. |  |  |
| 8.1 | Operational Controls | Implement risk controls and safe work procedures. |  |  |
| 8.2 | Emergency Preparedness | Prepare and test emergency response plans. |  |  |
| 9.1 | Performance Evaluation | Monitor and measure OHS performance. |  |  |
| 9.2 | Internal Audit | Conduct periodic safety audits. |  |  |
| 9.3 | Management Review | Evaluate OHSMS performance. |  |  |
| 10.2 | Incident & Nonconformity | Investigate incidents and take corrective action. |  |  |
| 10.3 | Continual Improvement | Foster continuous safety improvement. |  |  |
