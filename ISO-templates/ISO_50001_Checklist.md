#  ISO 50001:2018 – Energy Management System (EnMS) Checklist

**Purpose:**  
ISO 50001 sets requirements for implementing an Energy Management System that enables organizations to improve energy efficiency, reduce costs, and enhance environmental sustainability.

---

| Clause | Requirement | Description | Status (✓/✗) | Remarks |
|---------|--------------|-------------|---------------|---------|
| 4.1 | Context | Identify internal/external energy-related issues. |  |  |
| 4.2 | Interested Parties | Identify stakeholders related to energy management. |  |  |
| 5.1 | Leadership | Ensure top management commitment to energy performance. |  |  |
| 6.1 | Risks & Opportunities | Evaluate energy risks and improvement opportunities. |  |  |
| 6.3 | Energy Review | Analyze energy use, consumption, and baseline. |  |  |
| 6.4 | Energy Objectives | Establish measurable energy targets. |  |  |
| 7.2 | Competence | Ensure employees are trained in energy efficiency. |  |  |
| 8.1 | Operational Planning | Implement energy control measures. |  |  |
| 8.2 | Design & Procurement | Integrate energy efficiency in design and purchase. |  |  |
| 9.1 | Monitoring & Measurement | Track energy performance indicators (EnPIs). |  |  |
| 9.2 | Internal Audit | Conduct periodic energy audits. |  |  |
| 9.3 | Management Review | Review EnMS performance and energy goals. |  |  |
| 10.2 | Nonconformity | Correct deviations from EnMS procedures. |  |  |
| 10.3 | Continual Improvement | Enhance overall energy performance continuously. |  |  |
