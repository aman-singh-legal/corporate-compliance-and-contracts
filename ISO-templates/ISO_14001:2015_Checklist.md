| Step                               | Task                                                      | Status (✓/✗) |
| :--------------------------------- | :-------------------------------------------------------- | :----------- |
| **1. Context of the Organization** | Identify internal & external environmental issues.        |              |
|                                    | Determine interested parties and their needs.             |              |
| **2. Leadership**                  | Define and communicate the environmental policy.          |              |
|                                    | Assign roles and responsibilities.                        |              |
| **3. Planning**                    | Identify environmental aspects & impacts.                 |              |
|                                    | Assess compliance obligations (laws, permits).            |              |
|                                    | Set environmental objectives and targets.                 |              |
| **4. Support**                     | Ensure resources and competence.                          |              |
|                                    | Establish communication and awareness programs.           |              |
|                                    | Maintain documented information (EMS manual, procedures). |              |
| **5. Operation**                   | Implement controls for significant aspects.               |              |
|                                    | Prepare for and respond to emergencies.                   |              |
| **6. Performance Evaluation**      | Monitor and measure key parameters.                       |              |
|                                    | Conduct internal audits.                                  |              |
|                                    | Review performance with management.                       |              |
| **7. Improvement**                 | Take corrective action for nonconformities.               |              |
|                                    | Continually improve the EMS.                              |              |
